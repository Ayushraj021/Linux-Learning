-------------------------------------------------------------------------------file operations------------------------------------------------------------------
touch file_name: for creating a file.
nano file_name: editing of file.
mv file_name file_new_name: for renaming the file.
cp copyingfile_name where_to_copy_file_name: for copying the file content into another file.
rm file_name: to delete the file.
touch file_name1 file_name2: for creating multiple files.
rm *: to remove all the files.

-------------------------------------------------------------------------------directory operations------------------------------------------------------------------
mkdir directory_name: for creating a directory.
cd directory_name: to go into the directory.
ls -l: to check the files present in the directory.
rmdir directory_name: to remove the directory.
rmdir -r directory_name: for Removing the directory when it is not empty.
cd .. : exit from directory.
